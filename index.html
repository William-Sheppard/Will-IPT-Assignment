Vector based animation

Computer animation is the creation of digital image, which once put in sequntial order creates the movement of objects. I used Adobe animate to create a 2D animation using vector images. Vector images are usually shapes and are made up of equaitons connected together at points. Vector images have the cool ability to be shape tweened, meaning the object can be morphed as it is tweened, creating the squishing affect seen to the right.

3D animation

3D computer animation is the same as 2D computer animation in which you use computer generation to create images which once put in sequential order create movement. The difference is the introduction of a z-axis. I used blender to create this 3D animation. This was from a previous Yr 11 project in which I used rigging to fix the vertices to a bone strucutre in which I could tween those bones to create movement for the vertices. Go here to watch.
